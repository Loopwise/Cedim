Strings o cadenas de caracteres

Los String son objetos que representan una secuencia, cadena, arreglo o vector de caracteres, osea, representa una palabra o una oración.

Para poder usar las funciones de un string, se tiene que agregar la librería <string>.
Veamos como podemos operar con un string: 

#include <iostream>
#include <string>

using namespace std;

int main(){
    //Declaración
    string s;

    //Lectura
    getline(cin,s);//Veremos porque no cin<<

    //Recorrido
    // Recordemos que s es un VECTOR de caracteres, entonces: 
    for (int i = 0; i < s.size(); i++) {
    // s[i]: accede al elemento en la posicion i
    cout << s[i] << '\n';
    }

    // Imprimiendo todo el string
    cout << s << '\n';

    // Así podemos agregar caracter al final
    s += 'a'; 
    cout << s << '\n';

    //También podemos agregar una palabra o hasta una oración
    s +="hola"; 
    cout << s << '\n';

    // Chequear que hace esto
    cout << s.substr(2, 2) << '\n'; 

    // La comparación es igual que en otro tipo de dato
    if (s == "hola") cout << "ok" << '\n';

    return 0;

}

#include <iostream>
using namespace std;

int fibonnacci(int n){
    if( n==1 || n==2 ){
        return 1;
    }
    return fibonnacci(n-1)+fibonnacci(n-2);
}
int main(){
    int n; cin>>n;
    cout<<fibonnacci(n);

Introducción a la Fuerza Bruta

Hay una clase de problemas donde tienes algunas ecuaciones en m variables. En esas ocaciones, una primera buena idea es intentar una solución fuerza bruta. Para ello se encuentra el rango de valores que puede tomar cada variable, se hace un loop for cada una de ellas, se comprueba si las ecuaciones se satisfacen y se va almacenando las m-tuplas que satisfacen las condiciones. También, cuando estamos haciendo un for sobre una variable, estamos fijando su valor, asi dentro de cada for su valor se vuelve una constante, es decir, estamos reduciendo el número de variables. Por eso, usualmente m−1 for serán suficientes, pues la última variable suele ser fácil de calcular cuando las demás ya son constantes.
