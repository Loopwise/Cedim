¿Qué es un lenguaje de programación?
Un lenguaje de programación es un lenguaje formal que le proporciona a una persona, en este caso el programador, la capacidad de escribir una serie de instrucciones o secuencias de órdenes en forma de algoritmos con el fin de controlar el comportamiento físico y/o lógico de una computadora, de manera que se puedan obtener diversas clases de datos o ejecutar determinadas tareas. A todo este conjunto de órdenes escritas mediante un lenguaje de programación se le denomina programa.

Wikipedia-Lenguaje de programación

En este caso el lenguaje que vamos a aprender es C++ o también llamado C plus plus/Cpp.
¿Y por qué? Pues detallaremos algunas de sus características a continuación:

  • Programa orientado a objetos (POO).
  • Lenguaje muy didáctico, con este lenguaje puedes aprender muchos otros lenguajes con gran facilidad.
  • Es portátil y tiene un gran número de compiladores en diferentes plataformas y sistemas operativos.
  • Es un lenguaje de alto nivel.
Además, adeltantándonos un poco, C++ es un lenguaje sumamente potente, en comparación a otros lenguajes como python que realiza 10⁶ operaciones en un segundo, java que realiza aproximadamente 5⁹ operaciones, y C++ nos permite realizar 10⁸ operaciones en un segundo. Esto se detalla más a fondo en Programación Competitiva o Algoritmos (Pero para mayor información pueden revisar este link: Time complexity y sí quieres volar aún más te dejo este otro Competitive Programmer's Handbook, Antti Laaksonen: Cap 1-Cap 2 )

Entonces, ¿Qué necesito para aprender a programar?

Pues solo necesitas querer hacerlo, ya que es, como dice su propio nombre, Lenguaje de programación, es como aprender una lengua nueva como bien sería el inglés, solo que en este caso aprendes una lengua para comunicarte con una computadora y que esta te ayude en la resolución de problemas.
Y...¿Qué problemas podría resolver?
Puedes resolver cualquier problema que sea repetitivo y predecible, osea que tenga un patrón, sea un problema básico de cálculo, que consiste únicamente en seguir una secuencia de pasos, hasta un problema de Number Theory en el que probablemente tu única tarea sea encontrar ese patrón de alguna forma rebuscada o ingeniosa.

Entonces con esta pequeña introducción estamos listos para crear nuestro primer programa en C++.

Nuestro primer programa en C++: Hola Mundo 
#include <iostream>

int main(){
    std::cout<<"Hola Cedim";
    return 0;
}

Ahora vamos a explicar qué quiere decir cada parte:
El #include quiere decir que vamos a incluir nuestra libería llamada <iostream> que es la librería que vamos a utilizar para empezar en C++.
Después viene nuestra función principal que será int main() y dentro de ella estará nuestro objeto cout que nos servirá para imprimir un dato en pantalla (cout:Console Output).

¿Ahora qué pasa si queremos imprimir un "Adiós mundo"? Vamos a verlo en código: 

#include <iostream>

int main(){
    std::cout<<"Hola cedim";
    std::cout<<"Adios cedim";
}

Podemos ver que sí lo ejecutamos nos quedaría algo así:

Hola cedimAdios cedim

Y esto es porque no le hemos indicado a la computadora que tiene que darnos un salto de línea, en este caso tendríamos que indicarle con un \n, veámoslo: 

#include <iostream>

int main(){
    std::cout<<"Hola cedim\n";
    std::cout<<"Adios cedim";
    return 0;
}

Y la salida sería algo así:

Hola cedim
Adios cedim

Y ya para finalizar esto se podría realizar en una sola línea, poniendo todo en una misma " ", o separando mediante el "operador" <<, veámoslo: 

#include <iostream>

int main(){
    std::cout<<"Hola cedim\nAdios cedim\n";
    std::cout<<"Hola cedim\n"<<"Adios cedim";
    return 0;
}

Cuya salida sería:

Hola cedim
Adios cedim
Hola cedim
Adios cedim

Pero es demasiado molestoso el tener que poner std:: a cada rato si queremos usar nuestro tan apreciado cout, ¿Verdad? Por suerte los desarrolladores pensaron en eso y nos dieron la poderosa using namespace std; que nos servirá para llamar a los objetos de la std sin la necesidad de tener que poner ese molesto std:: como ya se había mencionado, además de darnos una nueva herramienta para dar un salto de línea: El endl; veamos como se vería el código usando endl y using namespace std; : 

#include <iostream>
using namespace std;
int main(){
    cout<<"Hola cedim"<<endl<<"Adios cedim";
    return 0;
}

Hola cedim
Adios cedim

Manejo de Bits

Los lenguajes de programación manejan operaciones (en general mas rápidas) para manipular los bits de los números enteros, entre ellas estan:

  • Desplazamiento de bits y negación: Permite usar un número entero y desplazar sus bits hacia la derecha o izquierda una cantidad de pasos o tomar los bits de un número e invertirlos.
  • Operaciones BITWISE: Operaciones logicas AND, OR y XOR.

Desplazamiento de bits y negación

El desplazamiento de bits se puede ver más como una multiplicación o división por una potencia de 2, su implementación es la siguiente

int x = 10; //    000000000000000000000000000001010 Representacion de todos sus 32 bits
x = (x << 10); // 000000000000000000010100000000000 Desplazamiento de bits hacia la izquierda
int y = 20; //    000000000000000000000000000010100 Representacion de todos sus 32 bits
y = (y >> 10); // 000000000000000000000000000000000 Desplazamiento de bits hacia la derecha

Además, se pueden invertir los bits de un número entero usando lo siguiente:

int x = 1000; // 000000000000000000000001111101000 Representacion de todos sus 32 bits
x = ~x; //       111111111111111111111110000010111 Se invierten los bits del número

Operaciones BITWISE

BITWISE AND

Esta operación toma dos argumentos a y b para realizar en cada par de bits en la misma posición la función lógica Y:

Su implementación es:

int x = 3;  // 011 Representación binaria
int y = 4;  // 100 Representación binaria
int z = x & y // 000 Resultado de BITWISE AND

BITWISE OW

Esta operación toma dos argumentos a y b para realizar en cada par de bits en la misma posición la función lógica O:

Su implementación es:

int x = 7;  // 111 Representación binaria
int y = 4;  // 100 Representación binaria
int z = x | y // 111 Resultado de BITWISE AND

BITWISE XOR

Esta operación toma dos argumentos a y b para realizar en cada par de bits en la misma posición la función lógica O EXCLUSIVO:

Su implementación es:

int x = 10;  // 1010 Representación binaria
int y = 4;  // 0100 Representación binaria
int z = x ^ y // 1110 Resultado de BITWISE AND

Entrada y Salida de datos

Lo que es la salida de datos ya lo vimos con el cout<< ahora nos falta ver una parte importante, que veremos en un pequeño problema usando un poco lo que ya hemos visto:

Queremos que un usuario nos digite dos números y en pantalla nos imprima todas las operaciones que se puedan realizar con estos dos números, hablando de operaciones aritméticas:

Veamos, el problema nos pide que de alguna forma el usuario nos digite dos números, y... ¿Cómo hacemos ello?. Pues de nuevo, la poderosa std nos da la respuesta: cin>> es un objeto que nos permite la lectura de datos, veámoslo en código resolviendo este problema:

Código

#include <iostream>
using namespace std;
int main(){
    int a,b;
    cout<<"Ingrese dos valores:";
    cin>>a;
    cin>>b;
    cout<<"La suma es: "<<a+b<<endl;
    cout<<"La resta es: "<<a-b<<endl;
    cout<<"La multiplicación es: "<<a*b<<endl;
    cout<<"La division es: "<<a/b<<endl;
    cout<<"El resto es: "<<a%b<<endl;
}
